Dash Callback Tetikleme Test Uygulaması

Aşağıdaki Dash uygulaması, buton tıklamasıyla bir Store'u, o Store'un diğerini, onun da alttaki yazıyı güncellemesini test eder.

{% plotly_app name=dash_app_name %}

Test Adımları:

  1. Sayfa yüklendiğinde terminal loglarına bakın. Sadece başlangıç logları olmalı, callback 1, 2, 3 tetiklenmemeli.
  2. "Değeri Artır" butonuna tıklayın.
  3. Terminal loglarına bakın. Callback 1 -> Callback 2 -> Callback 3 zincirinin tetiklendiğini gösteren logları sırayla görmelisiniz.
  4. Sayfadaki "Burada Store B değeri görünecek." yazısının güncellenip güncellenmediğini kontrol edin.
  5. Butona birkaç kez daha tıklayıp logları ve yazının tekrar güncellenip güncellenmediğini izleyin.
